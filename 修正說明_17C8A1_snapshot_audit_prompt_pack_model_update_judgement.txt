第 17-C-8A-1：單次快照稽核同步打包提示詞 + AI 判斷是否需更新產業模型

本次完成內容：
1. 買進決策版打包提示詞新增：
   - 產業模型建置時間 / 版本
   - 主分類原始 floor / soft / hard
   - 混合產業權重
   - 混合後 base / floor / soft / hard
   - 17-C-8A 單次快照稽核結果
   - 是否需人工檢查產業估值模型的判斷要求

2. 研究完整版打包提示詞新增：
   - 產業模型單次快照稽核與更新判斷完整摘要
   - 稽核結果、分數、系統建議動作
   - 目前 primary_taxon / hybrid_taxons / 混合後倍率
   - 現價、法人均價、法人高標隱含 Forward P/E
   - 系統 hard ceiling
   - 正面支持因素與風險因素
   - 歷史紀錄狀態提醒

3. 新增對外部 AI 的明確判斷要求：
   外部 AI 需從下列 5 種回答：
   - 不建議更新模型
   - 暫時觀察
   - 建議檢查 hybrid 權重
   - 建議檢查 primary_taxon
   - 建議檢查整個產業倍率

4. 防呆規則：
   - 單次快照不能當成長期重估證據。
   - 不可因現價高於 hard ceiling 就直接調高模型。
   - 若建議更新，必須說明是要改 primary_taxon、hybrid_taxons，還是 base / soft / hard。
   - 若不建議更新，必須說明原因是市場過熱、資料不足、法人樣本不足，或基本面尚未落地。

5. 模型版本字樣更新為 17-C-8A-1。
