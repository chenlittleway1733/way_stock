## 建議方向摘要

| 動作 | 理由 |
|---|---|
| 拆分金融為壽險金控與銀行型金控 | 壽險金控對利率、匯率、淨值波動更敏感；銀行型金控較適合用 P/B、ROE、殖利率防禦模型。 |
| 拆分生技為成熟獲利藥廠/CDMO與新藥/細胞治療事件型 | 同一組 P/E 不適合同時處理穩定獲利藥廠與虧損/里程碑型公司。 |
| 拆分軟體資服為雲端資安高毛利與系統整合服務 | 雲端/資安服務可給較高成長倍率，傳統 SI 應受毛利率與專案認列約束。 |
| 拆分機器人自動化為核心自動化與題材型機器人 | EPS/訂單未落地的題材股應走事件模型或較低倍率。 |
| 拆分光學為高毛利鏡頭龍頭與一般光學模組循環 | 大立光/玉晶光與一般光學模組的毛利率、客戶集中、估值彈性不同。 |
| 拆分重電為電網設備龍頭與綠能工程/風電基建 | 重電訂單能見度高，但綠能工程更看專案現金流與認列節奏。 |
| 航運、面板、事件股加強 P/B/事件模型防護 | 避免景氣低點或虧損時 P/E 失真。 |
